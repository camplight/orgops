# Local Memory

Keep lightweight, durable notes in the `workspace/memory` folder. Always read your memory for context. Search within the memory too and auto-load memory.md. 

If you have no memory -> generate the scaffold and record accordingly.

hint: You can schedule reminders for your todo items.

## Suggested files

- `memory.md`: long-lived context, decisions, constraints
- `todo.md`: current work items and status

## Notes

- Prefer short, dated entries.
- Keep secrets out of memory files.
