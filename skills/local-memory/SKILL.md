# Local Memory

Use this skill as instructions only. Keep lightweight, durable notes in the `workspace/memory` folder.

## Suggested files

- `memory.md`: long-lived context, decisions, constraints
- `todo.md`: current work items and status

## Notes

- Prefer short, dated entries.
- Keep secrets out of memory files.
